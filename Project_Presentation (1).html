LifeBuddy

Your Autonomous Life Concierge

Google AI Intensive Capstone • Track: Concierge Agents
Authors: Naveen Kumar Kaluvala & Charitha Miryala

The Challenge

  • Decision Fatigue: Balancing urgent deadlines, household tasks, and scheduling leads to cognitive fatigue.
  • Passive Task Tools: Traditional organizer apps require manual organization and do not help with actual execution.
  • Concierge Solution: The need for an active assistant that automates triage, prioritizes, and safely handles logistics on behalf of the user.

System Architecture

Decoupled multi-tier design: A local React Frontend served dynamically via a FastAPI backend orchestrator. High-performance background AI tasks utilize PostgreSQL/pgvector and MCP interfaces.

Core Components

ADK Orchestration

DAG-based message bus coordinating agent handoffs and context states.

Planner Agent

NLP-based parser that automatically routes tasks to the Eisenhower Matrix.

Shopper Agent

pgvector vector store similarity matches templates for proactive list building.

MCP Integrations

Secure tool interfaces executing direct calendar and email updates.

SDLC Conventions

  • Environment Separation: Configuration variables isolate Development, Staging, and Production layers via standard deployment.yaml files.
  • Local Sandboxing: All changes are deployed and verified on local execution environments before cloud deployment.
  • High Quality Standards: Gherkin specifications align features across development, QA, and deployment tiers.

Multi-Agent Optimization

  • 7 Specialized Roles: Separation of roles (Product Owner, QA, UX, Dev, etc.) models professional team workflows.
  • Artifact Handoffs: Strict interface contracts govern the transfer of specs, plans, and testing scripts between agents.
  • Minimized Context Windows: Narrow, task-oriented context scopes prevent model rot, optimize token consumption, and eliminate code hallucinations.

Context Integrity

  • AGENTS.md (Static Context): Houses global rules, directory indexes, core technical stacks, and security boundaries.
  • WORKSPACE_MEM.md (Dynamic Context): Acts as the dynamic sprint board tracking tasks, execution progress, and messaging logs.
  • Zero Ambient Authority: Prevents access credential creep by granting agents just-in-time tokens.

Progressive Skills & MCP

  • MCP Server Nodes: Connections like spec-repository-mcp and knowledge-base-mcp securely bind agents to codebases.
  • On-Demand Skills: Progressive disclosure rules load instructions (e.g. data-autocleaning) only when triggered by matching tasks.
  • Efficiency: Keeps prompt context clean and focused on task execution.

BDD Specifications

Gherkin specs act as the executable bridge between business rules and automated testing:
Feature: Predictive Household Shopping List
  Given a user's purchase history table in PostgreSQL
  When they prompt the Shopper Agent for "Summer BBQ"
  Then the agent queries pgvector semantic similarity
  And populates the shopping list with corresponding items

Zero-Trust Security

  • Structural Gate: SPIFFE role verification blocks unauthorized tool execution.
  • Semantic Gate: Scans payloads for PII, secrets, and credentials.
  • HITL Review: High-stakes actions require manual logic reviews.

Core User Journeys

We mapped out and verified the core product interactive journeys. Press the down arrow key or swipe down to see the real-world flow screenshots.

1. Task Buddy

Natural Language task creation. Instantly parses deadlines, scheduling commands, and priority instructions, automatically allocating them to the correct Eisenhower quadrants on submission.

2. Timeline Buddy

Provides visual calendar views by Day, Week, and Month. Integrated with the Planner Agent to support natural drag-and-drop scheduling transitions.

3. Groceries Buddy

Predictive shopping list building. Vector store template matching loads recipes and templates semantically, with proactive historically-based purchase recommendations.

4. Security Gate

Human-In-The-Loop gating. Triggers a logic review overlay when high-stakes calendar MCP operations are initiated, ensuring zero-trust verification.
